Google.comへのリンク集（様々な開き方パターン）

1. 通常のHTMLリンク（_self - 同じタブで開く）
Google（同じタブで開く）
1-2. 通常のHTMLリンク（openExternalBrowser=1を付与）
Google（同じタブで開く）
2. 通常のHTMLリンク（_blank - 新しいタブで開く）
Google（新しいタブで開く）
3. JavaScript - location.href（同じタブで開く）
4. JavaScript - location.assign（同じタブで開く）
5. JavaScript - location.replace（同じタブで開く、履歴を残さない）
6. JavaScript - window.open（新しいタブで開く）
7. JavaScript - window.open + _blank（新しいタブで開く）
8. JavaScript - window.open + _blank + noreferrer（新しいタブで開く）
9. JavaScript - window.open + 新しいウィンドウで開く
10. form要素とsubmitを使用
11. form要素とsubmitを使用（新しいタブ）
12. setTimeout を使用して遅延リダイレクト
13. iframe内で開く
